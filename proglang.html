Programming Languages

Term Definition
Machine Language a computer programming language consisting of binary or hexadecimal instructions which a computer can respond to directly.
Debugging the process of identifying and removing errors from computer hardware or software
Assembler The Assembler is a Software that converts an assembly language code to machine code
Assembly Language An assembly language is a programming language that communicates with the hardware of a computer directly
Translator A translator is a program. that converts source code into object code
Compiler a software tool that translates Python code written in high-level human-readable form into low-level machine code or bytecode that can be executed directly by a computer
Interpreter A python interpreter is a computer program that converts each high-level program statement into machine code
Cross-Platform Software Cross-platform software means a language can run equally on variety of platforms- Windows, Linux/UNIX, Macintosh, Supercomputers, Smart phones etc
Natural Language a subfield of artificial intelligence (AI) that deals with the interaction between humans and computers using natural language
Formal Language a language designed for use in situations in which natural language is unsuitable, as for example in mathematics, logic, or computer programming
Machine Independence Able to operate on many different types of equipment.
High level programming language A high-level programming language has a significant abstraction from the details of computer operation. It is designed to be easily understood by humans and for this reason they must be translated by another software.
Object-oriented programming a style of programming characterized by the identification of classes of objects closely linked with the methods (functions) with which they are associated.
scripting language A scripting language is a programming language that employs a high-level construct to interpret and execute one command at a time.
Data Types: Integer: a whole number, positive or negative, without decimals, of unlimited length. Float: a function or reusable code in the Python programming language that converts values into floating point numbers. String: a collection of alphabets, words or other characters
Primitive data type These data structures contain simplified data values and serve as the foundation for manipulating data.
Data Strucutre a way of organizing and storing data so that they can be accessed and worked with efficiently
Python Collection data structures: List: Lists are used to store data of different data types in a sequential manner.. Tuple:Tuples are another standard sequence data type. The difference between tuples and list is that tuples are immutable, which means once defined you cannot delete Set: A set is a collection of unique elements with no defined order, which are utilized when an object only needs to exist within a collection of objects dictionary: dictionaries (or dicts for short) are a central data structure
Function A function is a block of code which only runs when it is called
Local variable a local variable is a variable that is given local scope
Global variable a global variable is a variable with global scope, meaning that it is visible throughout the program, unless shadowed. The set of all global variables is known as the global environment or global state
Source program an original computer program written by a programmer that is converted into the equivalent object program, written in machine language, by the compiler or assembler
Object Program a computer program translated from the equivalent source program into machine language by the compiler or assembler.
